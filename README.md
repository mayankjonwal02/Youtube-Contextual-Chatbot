
## ğŸ”Œ API Routes

### `/api/transcript` (POST)

**Body**:

```json
{
  "videoUrl": "https://www.youtube.com/watch?v=xyz"
}
```

**Returns**:

```json
{
  "transcript": "This is the full transcript text..."
}
```

---

### `/api/chat` (POST)

**Body**:

```json
{
  "prompt": "What are the main takeaways?",
  "transcript": "Full transcript text here..."
}
```

**Returns**:

```json
{
  "response": "The main takeaways are..."
}
```
